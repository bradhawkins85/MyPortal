Webhook

Have your backup script POST to:

{{ backup_status_url }}

with this JSON body:

{
  "job_id": "{{ backup_editing_job.token }}",
  "status": "pass",
  "message": "Optional human readable message"
}

Accepted status values: pass, fail, warn, unknown (aliases such as ok, error, warning also work).

PowerShell example 
$body = @{ job_id = "{{ backup_editing_job.token }}"; status = "pass"; message = "Backup OK" } | ConvertTo-Json
Invoke-RestMethod -Uri "{{ backup_status_url }}" -Method Post -Body $body -ContentType "application/json"
