About CIS Benchmarks

The CIS Microsoft 365 Foundations Benchmark and associated Intune benchmarks provide consensus-based security configuration guidance. These checks use the Microsoft Graph API to evaluate your tenant configuration against a subset of the recommended controls.

Some checks require additional Graph API permissions (Policy.Read.All, Organization.Read.All, DeviceManagementConfiguration.Read.All, DeviceManagementManagedDevices.Read.All, AuditLog.Read.All). Re-provision the enterprise app to grant these permissions if checks show as unknown due to missing access.

Checks that are not applicable to your environment (e.g. built-in .onmicrosoft.com domains that cannot have DKIM configured) can be excluded using the Exclude button next to each failed or unknown check. Excluded checks are shown with a ⊘ icon and do not count as failures. Exclusions can be removed at any time to re-enable the check.
