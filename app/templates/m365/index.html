Mailbox report privacy check

Microsoft 365 can conceal mailbox identifiers in usage reports when the admin centre privacy option Display concealed user, group, and site names in all reports is enabled. When active, mailbox sync will fail because real user addresses cannot be identified.

Use this check to confirm whether the privacy setting is interfering with mailbox sync. If the check reports concealment, sign in to the Microsoft 365 admin centre, go to Settings → Org settings → Services → Reports and disable Display concealed user, group, and site names in all reports, then re-run the check.

How synchronisation works

  1. Store a tenant ID, client ID, and client secret for the company (or use Auto-provision if available).
  2. Authorize the application using the "Authorize portal access" button to grant offline access.
  3. Trigger a sync from the Licenses tab to import subscribed SKUs and assigned users.

Scheduled tasks can also call the synchronisation service to keep allocation data aligned with Microsoft 365.
